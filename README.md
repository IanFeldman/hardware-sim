# hardware-sim

Logic gate simulator - still under development.

## Controls
Mode is shown in the top right
### Simulate Mode
| Key | Description |
| --- | --- |
| `click 'Add Node'` | Add a node of specified type; Enter Place Mode |
| `i` | Toggle Inspect Mode |
| `p` | Toggle Pause Mode |

### Place Mode
| Key | Description |
| --- | --- |
| `w` | Move node up |
| `a` | Move node left |
| `s` | Move node down |
| `d` | Move node right |
| `enter` | Place node |
| `delete` | Cancel node placement |

### Inspect Mode
| Key | Description |
| --- | --- |
| `w` | Move cursor up |
| `a` | Move cursor left |
| `s` | Move cursor down |
| `d` | Move cursor right |
| `enter` | Give info on node |
| `delete` | Delete node |
| `i` | Toggle Inspect Mode |

### Pause Mode
| Key | Description |
| --- | --- |
| `p` | Toggle Pause Mode |
